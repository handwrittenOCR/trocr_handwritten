### 2. Paragraph splitter prototype (abandoned)
- Created `trocr_handwritten/parse/paragraph_splitter.py` — image-based paragraph splitting using morphological text extraction + horizontal projection + indentation detection
- Column detection (left/right page spread) worked well using vertical projection valley detection
- Line detection was too coarse for the very dense handwriting in gosier/1840 — lines merge together
- **User feedback**: line-level detection unnecessary; OCR quality from existing YOLO crops is fine
- Approach abandoned in favor of text-based post-OCR splitting

### 3. Dataset deduplication + registry splitting (implemented)
- **Problem**: YOLO produces overlapping crops (small strips contained inside full-column detections), and each crop contains many registry entries (each starting with "Aujourd'hui")
- **Solution**: Modified `trocr_handwritten/ner/dataset.py` with two new features:
  1. **`deduplicate_crops()`**: Detects when a small crop is >80% contained inside a larger one and removes the redundant smaller crop
  2. **`split_registries()`**: Splits transcription text on "Aujourd'hui" pattern to produce one ActRecord per civil registry entry
- Both features are backward-compatible — existing communes with Marge crops and "L'An mil huit cent" preambles are unaffected
- **Test results on gosier/1840**: 6 redundant crops removed, 108 individual registry entries extracted from 5 pages
- All entries are act_type "unknown" (no Marge crops) — NER will determine type from text content
