### 4. Fixed reading order sort: page-side interleaving bug
- **Bug**: `build_reading_order()` in `parse/utils.py` sorted all entries by y_center globally, interleaving left and right page crops. This caused cross-page-side continuation merges (~10% error rate)
- **Fix**: Sort by page side first (left page fully, then right page), then top-to-bottom within each side
- 1,501/1,668 pages had different reading order after this fix

### 5. Fixed preamble detection: multiple missing variants
- **"Aujourd'hui"**: Not recognized as new act start. Added `_AUJOURDHUI_PATTERN` check. Fixed gosier/1840, anse_bertrand/1840-1842.
- **Garbled OCR**: "L'An Aujourd'hui mil huit cent quarante mil trois" — OCR inserts words randomly. Added loose preamble match (`L'An` in first 50 chars + `mil` in first 100). Fixed deshaies/1843 (11→56 acts).
- **Strikethrough markers**: `~~Aujourd'hui~~ l'an mil huit cent` — stripped `~~...~~` before checking. Fixed deshaies/1840.
- **"Mêmes jour et an que dessus"**: Shorthand preamble in sainte_anne. Added `_MEMES_JOUR_PATTERN`. Fixed sainte_anne/1839 (159→222), sainte_anne/1840 (157→228).
- **"Pardevant nous"** at start: Act opening without date preamble. Added check.
- **"Mil huit cent"** without "L'An" prefix: Added fallback match.

### 6. NER regex extractor: act_type + Marge fields
- `detect_act_type_from_marge()`: keyword search (décès/naissance/mariage)
- `detect_act_type_from_plein_texte()`: formulaic indicators (décédé/accouchée)
- `detect_act_type()`: combined, Marge priority
- `extract_name_from_marge()`: person name after "de/d'"
- `extract_owner_from_marge()`: owner after au Sr/Sᵗ/Sʳ, Ve/Vve, bon/h^on, Héritiers
- `extract_act_number_from_marge()`: N° XX
- All fields added to `NERResult` schema: `marge_act_type`, `marge_act_name`, `marge_act_number`, `marge_act_owner`
