## Session 2: NER Model Comparison & Marge↔Plein Texte Fix

### Objective
Pick up NER extraction for transcribed pages, compare NER models, fix Marge↔Plein Texte pairing bug.

### Steps

1. **Assessed current state**:
   - OCR transcription complete for 5 communes (~13,000+ crops)
   - NER done only for abymes/1842: regex 228/228, LLM only 5/228 (stopped due to rate limiting)
   - No NER started for other communes

2. **Model research for NER**: Compared all available models across Gemini, OpenAI, Mistral for text extraction:
   - Current NER model: `gemini-3-flash-preview` ($0.50/$3.00 per M tokens)
   - Cheapest viable: `gemini-2.5-flash-lite` ($0.10/$0.40) — 5x cheaper
   - Also considered: GPT-5.4-nano, Mistral Small 3.1
   - Note: `gemini-3-pro-preview` (OCR model in CLAUDE.md) is **deprecated** since March 9, 2026

3. **NER model comparison** on 20 acts from abymes/1842:
   - Ran `gemini-3-flash-preview` (16 new + 4 existing) and `gemini-2.5-flash-lite` (20 acts)
   - Cost: Flash EUR 0.0168 (16 acts), Flash-Lite EUR 0.0031 (20 acts) — **5.4x cheaper**
   - 77.2% agreement rate between models
   - Most "mismatches" are formatting differences (Flash keeps verbatim text, Flash-Lite normalizes dates/times/ages to numbers)
   - Flash-Lite extracted more fields overall, especially officer_name
   - Output: `logs/ner_model_comparison_2026-04-03.md`

4. **Discovered Marge↔Plein Texte pairing bug**:
   - Scans are **double-page spreads** (left page x < ~3000, right page x > ~3000)
   - `build_reading_order()` in `parse/utils.py` matched by y-overlap only, crossing page boundaries
   - **74% of all pages had wrong pairings** (856/1151)
   - NOT a cross-page continuation issue (those are rare, ~3%)

5. **Fixed pairing with page-side awareness**:
   - Added x-coordinate thresholds: Marge x < 1500 = left page, Plein Texte x < 3000 = left page
   - Only match Marge and Plein Texte on the same page side, then y-overlap within
   - Validated on pages 005-007: 44% correct → 100% correct
   - Content-verified: person names from Marge found in paired Plein Texte

6. **Saved corrected reading order as separate file**:
   - Original `metadata.json` left untouched
   - Fixed reading order saved as `metadata_reading_order.json` per page folder
   - 1,151 files created across all communes/years
