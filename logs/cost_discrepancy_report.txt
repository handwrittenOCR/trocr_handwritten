ROOT CAUSE

They are billed as output tokens but NOT reported in the
completion_tokens field of the OpenAI-compatible endpoint.

KEY ISSUE
---------
The OpenAI-compatible endpoint (generativelanguage.googleapis.com/
v1beta/openai/chat/completions) returns a usage object with
prompt_tokens and completion_tokens. These fields do NOT include
thinking tokens. There is no other field reporting thinking usage
in this endpoint. Developers using this endpoint have no way to
monitor actual billed costs in real time.
